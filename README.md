# Enyr Website

Modern static HTML website for Enyr - Data Analytics & IoT Solutions for Energy Markets

## Features

- Responsive design for all devices
- Modern teal/turquoise color palette matching the Enyr logo
- Smooth animations and interactive elements
- Sections for Services, Solutions, About, and Contact
- SEO-friendly semantic HTML

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete responsive styling
- `script.js` - Interactive functionality
- `logo.png` - Enyr logo

## Deployment Options

### Option 1: GitHub Pages (Recommended - Free)

1. Create a new repository on GitHub
2. Push this code to the repository
3. Go to repository Settings > Pages
4. Select "main" branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Option 2: Netlify (Easy drag-and-drop)

1. Go to [netlify.com](https://www.netlify.com/)
2. Sign up for a free account
3. Drag and drop this folder to deploy
4. Get instant public URL

### Option 3: Vercel (Fast deployment)

1. Go to [vercel.com](https://vercel.com/)
2. Sign up for a free account
3. Import this repository or drag and drop
4. Get instant public URL

## Local Development

Simply open `index.html` in your browser to view the site locally.
